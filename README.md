# Kubernetes Lab с мониторингом и API

## Описание проекта
Лабораторный стенд для изучения Kubernetes, мониторинга (Prometheus + Grafana) и API-интеграции.

## 🚀 Быстрый старт

### Требования
- Windows 10/11 Pro/Enterprise
- Docker Desktop с включенным Kubernetes
- Helm
- Chocolatey (опционально)

### Установка
```powershell
# Клонировать репозиторий
git clone <your-repo-url>
cd k8s-lab

# Запустить стенд
.\start.ps1
